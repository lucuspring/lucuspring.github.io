书接上文“Digital-Ocean上科学上网”，第一个充分条件是“拥有一个科学的服务器”，本篇从①注册账户，②开通服务两个部分介绍。

0x00 注册账户

创建一个PayPal账户

1.访问到PayPal主页

2.注册一个人账户

3.填写具体用户信息

4.登录账户

5.绑定银行卡

6.选择信用卡或者借记卡

初始资金激活

1.访问DigitalOcean登录界面

2.选择PayPal方式激活账户，需要预支5刀

3.跳转到PayPal的付款界面

4.验证绑定邮箱验证码

0x01 开通服务

1.可以查看个人账户，这里多出的50刀是Github Education返现的，详情参考“How to get github pack”篇

2.在Droplets中选择需要的服务类型

3.在Droplets管理中查看各种服务器类型和参数
